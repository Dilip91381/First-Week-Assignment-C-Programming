# 📱 Mobile Recharge GST Calculator (C)

A simple C program that calculates the **18% GST** on a mobile recharge amount and displays the **final payable amount**.

---

## 📌 Features

- Accepts recharge amount from the user.
- Calculates **18% GST**.
- Displays:
  - GST amount
  - Total amount payable
- Simple and beginner-friendly C program.

## ▶️ How to Run

### Using GCC

1. Compile the program:

```bash
gcc recharge_gst_calculator.c -o recharge
```

2. Run the program:

```bash
./recharge
```

### On Windows

```bash
recharge.exe
```

---

## 💻 Sample Output

```
Enter Recharge Amount: 500
GST = 90.00
Total = 590.00
```

---

## 🧮 Formula Used

```
GST = Recharge Amount × 18 / 100

Total Amount = Recharge Amount + GST
```
