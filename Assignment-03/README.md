# Assignment 03 - Pointer Playground

## Objective

To understand the basic concepts of pointers in C programming.

## Problem Statement

Write a menu-driven C program to demonstrate pointer concepts like displaying memory addresses, dereferencing pointers, swapping numbers using pointers, traversing an array using pointers, and pointer arithmetic.

## Algorithm

1. Display the menu.
2. Select an option.
3. Perform the selected pointer operation.
4. Display the result.
5. Repeat until Exit.
6. Stop the program.

## Time Complexity

- Display Address : O(1)
- Dereference Pointer : O(1)
- Swap Numbers : O(1)
- Array Traversal : O(n)
- Pointer Arithmetic : O(n)

## Space Complexity

O(n)

## Sample Input

```
10
20
```

## Sample Output

```
Before Swapping

10
20

After Swapping

20
10
```

## Learning Outcome

- Learned pointer declaration.
- Understood pointer dereferencing.
- Learned pointer arithmetic.
- Traversed arrays using pointers.
- Swapped values using pointers.
