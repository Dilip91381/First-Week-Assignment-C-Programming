# 🔐 Character Encoding Utility (C)

A simple C program that performs **character encoding** by converting input characters into encoded values using a basic encoding technique. This project demonstrates string manipulation, character processing, and fundamental programming concepts in C.

---

## 📌 Features

- Accepts text input from the user.
- Encodes characters using a simple encoding algorithm.
- Displays the encoded output.
- Beginner-friendly and easy to understand.
- Lightweight and fast.

## 🧮 Algorithm

1. Read the input string from the user.
2. Process each character one by one.
3. Apply the encoding rule.
4. Store the encoded characters.
5. Display the encoded string.
