# Assignment 06 - Inventory Management System

## Objective

To develop an Inventory Management System using structures in C.

## Problem Statement

Write a menu-driven C program to manage product records. The program should add, display, search, update, sort products by price, and calculate the total inventory value.

## Algorithm

1. Start the program.
2. Display the menu.
3. Select an operation.
4. Perform the selected task.
5. Repeat until Exit.
6. Stop the program.

## Time Complexity

- Add Product : O(1)
- Display Products : O(n)
- Search Product : O(n)
- Update Product : O(n)
- Sort Products : O(n²)
- Total Inventory Value : O(n)

## Space Complexity

O(n)

## Sample Input

```
Product ID : 101
Product Name : Laptop
Price : 55000
Quantity : 2
```

## Sample Output

```
Product Added Successfully.

Total Inventory Value = 110000.00
```

## Learning Outcome

- Learned to use structures with arrays.
- Implemented searching and sorting.
- Calculated inventory value.
- Improved understanding of functions.
