# Assignment 04 - Student Record Management System

## Objective

To create a Student Management System using structures in C.

## Problem Statement

Write a menu-driven program to add, display, search, update, and delete student records using an array of structures.

## Algorithm

1. Start the program.
2. Display the menu.
3. Select an operation.
4. Perform the selected task.
5. Repeat until Exit.
6. Stop the program.

## Time Complexity

- Add Student : O(1)
- Display Students : O(n)
- Search Student : O(n)
- Update Student : O(n)
- Delete Student : O(n)

## Space Complexity

O(n)

## Sample Input

```
Roll No : 101
Name : Rahul
Marks : 85
```

## Sample Output

```
Student Added Successfully.

Roll No : 101
Name : Rahul
Marks : 85.00
```
