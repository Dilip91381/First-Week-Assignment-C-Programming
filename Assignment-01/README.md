# Assignment 01 - Student Marks Analyzer

## Objective

To develop a menu-driven C program that stores and manages the marks of students using arrays and functions.

## Problem Statement

Write a C program to store the marks of N students. The program should display marks, find the highest and lowest marks, calculate total and average marks, search for a student's marks, and sort the marks in ascending order.

## Algorithm

1. Start the program.
2. Display the menu.
3. Enter the number of students and their marks.
4. Perform the selected operation:
   - Display marks
   - Find highest marks
   - Find lowest marks
   - Calculate total marks
   - Calculate average marks
   - Search marks
   - Sort marks
5. Repeat until the user selects Exit.
6. Stop the program.

## Time Complexity

- Display Marks : O(n)
- Highest Marks : O(n)
- Lowest Marks : O(n)
- Total Marks : O(n)
- Average Marks : O(n)
- Search Marks : O(n)
- Sort Marks : O(n²)

## Space Complexity

O(n)

## Sample Input

```
Enter number of students : 5

Marks:
78
65
90
84
72
```

## Sample Output

```
Highest Marks = 90

Lowest Marks = 65

Total Marks = 389

Average Marks = 77.80

Sorted Marks:
65
72
78
84
90
```
