# Assignment 02 - String Utility Toolkit

## Objective

To perform different string operations using functions in C.

## Problem Statement

Write a menu-driven C program to perform string operations such as finding string length, reversing the string, checking palindrome, counting vowels, digits and spaces, and displaying character frequency.

## Algorithm

1. Start the program.
2. Enter a string.
3. Display the menu.
4. Perform the selected operation.
5. Repeat until Exit is selected.
6. Stop the program.

## Time Complexity

- Length : O(n)
- Reverse : O(n)
- Palindrome : O(n)
- Count Characters : O(n)
- Character Frequency : O(n²)

## Space Complexity

O(n)

## Sample Input

```
Hello World 123
```

## Sample Output

```
Length = 15

Reverse = 321 dlroW olleH

Palindrome = No

Vowels = 3

Digits = 3

Spaces = 2
```

## Learning Outcome

- Learned string handling in C.
- Understood loops and functions.
- Learned character counting and palindrome checking.
- Practiced string traversal using arrays.
