# Assignment 05 - Dynamic Array Management

## Objective

To learn dynamic memory allocation in C using malloc() and free().

## Problem Statement

Write a menu-driven C program to dynamically allocate memory for storing student marks. Display marks, find highest and lowest marks, calculate total and average, and release memory after use.

## Algorithm

1. Enter the number of students.
2. Allocate memory using malloc().
3. Enter student marks.
4. Perform the selected operation.
5. Free the allocated memory before exiting.
6. Stop the program.

## Time Complexity

- Enter Marks : O(n)
- Display Marks : O(n)
- Highest Marks : O(n)
- Lowest Marks : O(n)
- Total Marks : O(n)
- Average Marks : O(n)

## Space Complexity

O(n)

## Sample Input

```
Enter number of students : 4

Marks:
70
82
95
88
```

## Sample Output

```
Highest Marks = 95

Lowest Marks = 70

Total Marks = 335

Average Marks = 83.75
```

## Learning Outcome

- Learned dynamic memory allocation.
- Used malloc() and free().
- Practiced pointers with arrays.
- Improved understanding of memory management.
